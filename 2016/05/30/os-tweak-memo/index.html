Ubuntu 16.04 映射 CapsLock 为 Ctrl

  1. 指令：sudo vim /etc/default/keyboard, 将一行改为 XKBOPTIONS="ctrl:nocaps";
  2. sudo dpkg-reconfigure keyboard-configuration 一路Enter.

更改 Ubuntu BIOS 时间为本地时间

Ubuntu 默认把系统时钟设为格林尼治天文台标准时间，然后在此基础上加 8 小时。这会与其他并存的系统产生冲突。
timedatectl set-local-rtc 1
